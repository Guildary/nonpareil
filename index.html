nonpareil for os x

nonpareil for OS X is set of applications that simulate some of vintage HP calculators. Version provided for OS X is based on nonpareil 0.77 for Linux - a micro-assembler and simulator package written originally for Linux by Eric Smith.

nonpareil is made available under the terms of the Free Software Foundation's General Public License, Version 2.
